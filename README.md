# Fixit Lanka - Worker App

The Fixit Lanka Worker App represents the execution arm of the Fixit Lanka platform. Tailored for service professionals, this app enables workers to view their assigned tasks, manage their schedules, and update job statuses upon completion seamlessly.

## Features

- **Worker Authentication**: Secure login for registered professionals.
- **Assigned Jobs List**: View newly assigned and ongoing tasks with full details.
- **Status Updates**: Mark jobs as "In Progress", "Completed", or "Delayed".
- **Location Integration**: See job locations for quick navigation.
- **Real-time Sync**: Instant communication of job status back to the User and Supervisor apps.

## Technologies Used

- **Framework**: Flutter
- **Backend/Database**: Firebase (Authentication, Firestore Database)

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/ThEVI22/Fixit_Lanka-Workerapp.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Fixit_Lanka-Workerapp
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run the app:
   ```bash
   flutter run
   ```

## Setup

Please make sure the Firebase environments are synced properly by including `google-services.json` (Android) and `GoogleService-Info.plist` (iOS).
